# fake_type
以极快的速度将文件伪装成另一种格式，并无损恢复它们。

- [ ] 将很快提供第一版可执行文件

前往这个[网站](https://en.wikipedia.org/wiki/List_of_file_signatures)查找你需要的类型文件头 hex 值  然后将其 hex值转换为 u8 的数组 (tests中有实现)
